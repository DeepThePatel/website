OVERVIEW

This is a group project I completed in my Software Engineering class. The purpose of this project was to learn how to create a terminal-based Java application. We were assigned to create an application that replicated what a summer camp would use. This included the ability for different types of users to login: the director, counselors, and customers. Depending on the type of user logging in, there would be different functionalities.

DIRECTOR
  • View and create sessions
  • View list of counselors, guardians, and campers
  • Create and view camper schedules
  • View and edit camper information and health records
  • View and create activities
  • View and create cabins and assign campers
COUNSELORS
  • View camper schedules and information
  • View cabins
  • View activities
GUARDIANS
  • Sign up for an account
  • Input camper information (saves to JSON)

KEY FEATURES
  • JUnit Tests
  • Storing/Using data in JSON files
  • Utilizing a facade
