OVERVIEW

This is a video game I individually created in my MART 380 (New Media Art) class. The objective of this game is to navigate through the three areas of the map and find the key in each area. Once all three keys have been found, the player will then have to find the hidden portal and jump through it to win the game.

This game was built and designed in Unity and programmed in C#. Unfortunately I do not have the original files and code anymore, but the game is still playable by clicking the link above.
