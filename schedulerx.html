OVERVIEW

SchedulerX is a lifestyle Android app designed and produced by a team of five for our Senior Capstone Project. Using React-Native as the foundation of this app, it is aimed at helping users manage their schedules and track their fitness and nutrition goals.

The app is written in JavaScript using the React Native framework and uses Firebase for cloud storage and user authentication.

It has been tested through JUnit testing and behavioral testing using Detox. The demo video on how the tests run can be found here.



FEATURES
  • Create and manage tasks on your personal calendar
  • Track task and fitness progress
  • Browse a selection of workouts and meals — Powered by API Ninjas ↗
  • Personalize and share your own workouts and meals
  • Switch between Dark and Light Mode Themes
  • Safely secure personal information in Firebase storage
